# `keramics-datetime`

Date and time types for Keramics.

[docs.rs](https://docs.rs/keramics_datetime)

Supported date and time formats:

* 32-bit POSIX timestamp (time_t)
* 64-bit POSIX timestamp in nanoseconds (struct timespec)
* FAT date and time
* Windows FILETIME
